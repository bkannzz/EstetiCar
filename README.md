O projeto EstetiCar é um site multi-page desenvolvido com HTML, CSS e JavaScript, criado para apresentar os serviços de uma oficina especializada em estética automotiva. O objetivo do projeto foi oferecer uma experiência moderna e intuitiva para os clientes, com páginas organizadas para exibir informações sobre a empresa, serviços oferecidos, contatos e formas de atendimento. Além do design responsivo e visual atrativo, o site foi pensado para fortalecer a presença digital do cliente e facilitar a comunicação com os usuários.
